Les commandes git :

  • git init : permet d'initialiser le dossier actuel
  • git config –global user.email : permet d'enregistrer un email
  • git config –global user.name : permet d'enregistrer son nom
  • git config –global color.ui true :
  • git config –list : permet de voir tout ce qui a été défini
  • git status : permet de voir les dossiers référencés
  • git add ajoute les fichiers
  • git add *.html permet d'ajouter tous les fichiers avec l'extention .html
  • git commit -m « message » permet d'ajouter un commit de ce que l'on vient de faire
  • git log permet de voir tous les commits effectués
  • git log -n x : permet de voir les x derniers commits, exemple git log -n 3 permettra de voir les 3 derniers commits
  • git log –oneline : permet d'afficher les infos sur une seule ligne
  • git log « nom du fichier » : permet de voir toutes les modifications sur le fichier demandé
  • git log -n x -p « nom du fichier »: affiche les informations complètes du fichier, -n x permet aussi de filtrer les x dernières lignes
  • git checkout +clé: permet de revenir à l’état donné
  • git diff : permet de voir les modifications sur le fichier
  • git push -u origin master : permet d'envoyer les fichiers de ungit sur github

Mettre en ligne un site avec github pages:

Créer une branch « gh-pages » dans le repository de github dans le terminal taper « git branch gh-pages », puis faire un git checkout gh-pages, toujours dans le terminal faire un « git add *.* » et faire un « git commit -m » et pour finir faire un « git push ». Retourner sur github aller dans setting et recuperer le lien.
